# MA4012-Competition
This repository contains the code for our group 3's robot ***(Rollin')*** for the **MA4012 Competition**.

Click [here](https://www.youtube.com/watch?v=1H598Q3ScQc) to view youtube video for the competition.

The codes are written in *C* language, which are complied and downloaded onto VEX Cortex Microcontroller using [RobotC](https://www.vexrobotics.com/robotc-vexedr-vexiq.html?___store=vexroboticsap&___from_store=vexrobotics).


<ins>**C file used for competition**</ins>:
| File | Description |
|---|---|
| `main_competition.c` | main file used during competition |
| `var_competition.c` | variables and functions used during competition |
| `encoder_competition.c` | encorder task used during competition |

<br>

<ins>**Alternative libraries used** (Not applicable to `*competition.c` files)</ins>:
- [BNSBluetooth Library by JMarple](https://github.com/JMarple/BNSBluetooth) (For Testing - see debug stream on android devices)
